Licensing

The short version.

Plain language, so you know what you can do. Three licences — one each for the data, the media, and the code: the data and media are open, the code is source-available.

This is a prototype — a first draft of our intentions, not final or binding policy. The licences below describe the direction we intend for the Commons, in plain language — subject to change before launch.

The data

Open Database License 1.0 (ODbL) · contents DbCL 1.0
Use the data for anything — including commercial products — as long as you credit it, share back any improved database under the same licence, and keep it open. The same deal as OpenStreetMap.
You can
  • ✓ Use, copy, and share it — free
  • ✓ Adapt it and build products on it
  • ✓ Use it commercially
You must
  • ⚑ Attribute — credit the Cycling Commons + OpenStreetMap
  • ⚑ Share-alike — publish an adapted database under ODbL
  • ⚑ Keep it open — no DRM or lock-out of the data

The media

Creative Commons Attribution-ShareAlike 4.0 (CC BY-SA 4.0)
Use the photos and video for anything — including commercially — if you credit the creator and the Commons, and release any adaptations under the same licence.
You can
  • ✓ Use, copy, and share it — free
  • ✓ Adapt, crop, and remix it
  • ✓ Use it commercially
You must
  • ⚑ Attribute — credit the creator and the Cycling Commons
  • ⚑ Share-alike — release adaptations under CC BY-SA 4.0

The code

PolyForm Shield License 1.0.0 · source-available
Read, run, modify, and build on the Cycling Commons source for any purpose — except running a product that competes with the Cycling Commons. Want to build a cycling app? Build on the open data and API, free for any use including commercial.
You can
  • ✓ Read, run, and modify the source — free
  • ✓ Adapt it for a non-competing use (e.g. another sport)
  • ✓ Self-host and contribute back
You must
  • ⚑ Not compete — no product that competes with the Cycling Commons
  • ⚑ Keep notices — retain the licence and the Required Notice
